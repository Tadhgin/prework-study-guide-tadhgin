# Study Guide 

````
## User Story
As a coding bootcamp student,
I want a remote pre-work study guide, 
so I can keep & review all my work. 
````

## Acceptance Criteria

````
GIVEN a remote pre-work study guide
WHEN I visit the site in chrome
THEN I see 4 issues HTML; CSS; Git & JavaScript.
WHEN I view the study guie
THEN I see a dark blue header & footer, with 4 boxes & a shadow.
WHEN I open my repo on github
THEN I can see the issues listed with a suggestion on what I should study first.
WHEN I visit my repo with my URL
THEN I can access it from any browser.
````
